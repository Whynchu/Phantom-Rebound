# Boon Structure Ideas

1. **Tiered paths** – split boons into offensive, defensive, and utility lanes, and present one lane per room so players balance progression instead of grabbing duplicates.
2. **Combo modals** – group related effects (projectile spacing, speed, crit) into meta-boons and let later choices upgrade whichever meta path the player picked earlier.
3. **Guaranteed upgrade tiers** – early rooms grant basic stats (damage, firerate), mid-game unlocks shield/orbit tech, late-game gives high-cost effects such as auto-harvest or adaptive recovery.
4. **Limited duplicates** – swap the random pool for a small “draft” where the last pick has higher chance to repeat a category the player is weak in.
5. **Contextual assists** – add boons that trigger based on room difficulty (e.g., extra shields when audio says “room 10+”), keeping purple bullets locked to the harder enemies.

Use this doc as a reference when expanding the boon dataset so each slice (offense, defense, support) has clear ownership and synergy.
